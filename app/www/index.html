About

Use this visual tool for calculating molar mass for any chemical formula. Options for hiding the symbol or name of the elements provide a handy learning aid for memorizing the periodic table. Please note that the chemical formula is not limited to the known chemical compounds.

This periodic table is created by Dr Kaisa Helttunen and Heikki Kainulainen. Source code is available at GitHub.

iOS

We have also created a Periodic Table app for iOS.

References

Atomic weights are according to IUPAC © 2013 (Pure Appl. Chem., Vol. 85, pp. 1047-1078, 2013).

Revised values for aluminium, arsenic, beryllium, cadmium, caesium, cobalt, fluorine, gold, holmium, manganese, molybdenum, niobium, phosphorus, praseodymium, scandium, selenium, thorium, thulium and yttrium are taken from IUPAC website.

Conventional atomic weights are used for boron, bromine, carbon, chlorine, hydrogen, lithium, magnesium, nitrogen, oxygen, silicon, sulfur and thallium.

For radioactive elements with no stable isotopes, mass number of the most stable isotope is shown in brackets.
